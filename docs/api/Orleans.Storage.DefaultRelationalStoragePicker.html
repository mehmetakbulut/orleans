Class DefaultRelationalStoragePicker

A strategy to pick a serializer or a deserializer for storage operations. This can be used to: 1) Add a custom serializer or deserializer for use in storage provider operations. 2) In combination with serializer or deserializer to update stored object version. 3) Per-grain storage format selection 4) Switch storage format first by reading using the save format and then writing in the new format.
